Программа переводит списки работников отделов в XML формат.
Для обработки данных создайте в папке с программой текстовый файл
с названием workers.txt, формат заполнения -
Отдел:
						
	имя фамилия работника
						
	имя фамилия работника
					   
Отдел:

или перенесите нужный вам текстовый файл и переименуйте его
в workers.txt, результатом работы программы будет та же
информация в xml формате
<root>
<Отделы>
<Продавцы>
<Работник1>Иван Грачиков </Работник1>
</продавцы>
</Отделы>
</root>
